About the Project:
------------------
PHPCRM Open-Source is a powerful customer relationship management system built to boost business efficiency and accelerate sales growth.


This version is designed for developers and companies who want a CRM solution that is simple to customize, self-host, and extend according to their business requirements.

Key Modules Included:
---------------------
• Dashboard with key CRM metrics
• Lead Management
• Customer Management
• Tasks / Follow-up Scheduling
• Notes / Activity Logging
• Secure User Login
• Change Password

Technology Stack:
-----------------
• PHP 7.4+ (PHP 8+ recommended)
• MySQL / MariaDB
• MVC Architecture (Custom Pure PHP)
• Bootstrap 5 for UI
• mysqli (no external frameworks required)

Installation Summary:
---------------------
1. Extract the CRM source into your web directory (local or server).
2. Create MySQL database.
3. Import the installation SQL file from:
       /database/phpcrm_install.sql
4. Update database credentials in:
       /app/config/database.php
5. Access the CRM in browser:
       http://localhost/your-crm-folder/public/
